### 목적 조건

| 목적 | 지급 구조 | 설명 |
|------|-----------|------|
| 전환 | 단일 고정 | 신규 라이더 전환 유도. 가입일 이내 조건 자동 활성화 |
| 활성화 | 단일 / 단계 선택 | 기존 라이더 활성화 유도 |

### 전체 공통 조건

| 조건 | 필수 | 설명 |
|------|------|------|
| 기간 설정 | Y | 캠페인 시작일 ~ 종료일 |
| 미션 공개일 | Y | 라이더에게 미션이 노출되는 날짜 |
| 예산 | Y | 캠페인 총 예산. 소진 시 지급 중단 |
| 지역/ZONE | Y | 적용 지역 또는 ZONE. 다중 선택 가능 |
| 운영 방식 | Y | 정적 / 동적 |
| 화주사 | N | 전체(디폴트) / 로컬 / 브랜드 / 플랫폼 |
